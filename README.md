# IAs_Autolote-API-de-Predicci-n-de-Ventas-y-Recomendaciones-Financieras-con-Machine-Learning
Microservicio desarrollado en Python que utiliza técnicas de Machine Learning para predecir ingresos y volumen de ventas y generar recomendaciones financieras automatizadas para la toma de decisiones en negocios tipo autolote.
